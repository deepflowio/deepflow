# trident-rs

trident in rust

# X86 Dynamic Libraries

This project requires the following dynamic libraries:

1. **linux-vdso.so.1**
2. **libpthread.so.0**
3. **libz.so.1**
4. **libstdc++.so.6**
5. **libgcc_s.so.1**
6. **librt.so.1**
7. **libm.so.6**
8. **libdl.so.2**
9. **libc.so.6**
10. **ld-linux-x86-64.so.2**
